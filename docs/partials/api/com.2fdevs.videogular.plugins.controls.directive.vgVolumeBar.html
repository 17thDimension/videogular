vgVolumeBar
directive in module com.2fdevs.videogular.plugins.controls

Description

Directive to display a vertical volume bar to control the volume. This directive must be inside vg-volume directive and requires vg-mute-button to be displayed.

<videogular vg-theme="config.theme.url">
   <vg-video vg-src="sources"></vg-video>

   <vg-controls vg-autohide='config.autohide' vg-autohide-time='config.autohideTime'>
       <vg-volume>
           <vg-mute-button><vg-mute-button>
           <vg-volume-bar><vg-volume-bar>
       </vg-volume>
   </vg-controls>
</videogular>

Usage

as element:
<vg-volume-bar>
</vg-volume-bar>
