vgFullscreen
service in module com.2fdevs.videogular

Description

Native fullscreen polyfill service.

vgFullscreen.onchange: String with the onchange event name. vgFullscreen.onerror: String with the onerror event name. vgFullscreen.isAvailable: Boolean with fullscreen availability. vgFullscreen.isFullScreen: Boolean with current view mode. vgFullscreen.exit: Exit fullscreen function. vgFullscreen.request: Request for fullscreen access function.
